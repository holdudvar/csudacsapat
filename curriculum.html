A weboldalak építőkövei

A webes tartalmak változatos, stílusos és interaktív tartalmának megjelenítését az elmúlt évtizedekben úgy tették lehetővé, hogy különféle és folyamtosan fejlődő szabványokat igazítottak egymáshoz. Ezek a szabványok kódok "nyelvtanát" jelentik, a kódokat pedig a böngészőprogramok értelmezik. Ha minden rendben zajlik (helyes a kód, nincs vírus és gyors a netkapcsolat, stb.), akkor a böngészők villámgyorsan megjelenítik a felhasználó számára a weboldalt. Nézzük a három legfontosabbat kódkészletet!

1. HTML (angolul: HyperText Markup Language=hiperszöveges jelölőnyelv) A struktúráért felel. A weboldalak első betöltődő része rendszerint az index.html-nevű szöveges fájl, ami egyben a weboldal "Főoldala" is. Ide vannak beírva azok a "tag"-nek nevezett HTML elemek, amik az oldal tartalmáért felelnek. A tartalom alatt főleg a szöveget, felsorolásokat (listákat), táblázatokat és a fájlon kívüli tartalmakra vonatkozó hivatkozásokat értjük. A tag-ek egymásba ágyazódnak, ún. szülő-gyermek kapcsolat alapján. Más HTML fájlokat (pl. aloldalakat vagy más weboldalakat) is meghívhatunk, a rájuk mutató, kattintható szöveget pedig "hipertextnek" nevezzük. A képek nem részei a HTML szöveges fájloknak, viszont egyértelmű utasítás van arra, hogy hol keresse őket a böngésző. Leggyakrabban az index.html fájl mellett (ugyanabban a könyvtáran) egy "img" nevű mappában tároljuk őket.

2. CSS (angolul: Cascading Style Sheets = egymásba ágyazott stíluslapok) A megjelenítésért felel. A HTML elemek megformálására maga a HTML szabvány csak nagyon korlátozott mértékben alkalmas, ezért HTML-elemeket módosítjuk egy szorosan kapcsolódó, általában CSS nevű fájlban leírt kódkészlettel. A HTML elemek kiválasztók (szelektorok) segítségével hívják be a CSS-ből a rájuk vonatkozó formázási utasításokat.

3. JavaScript (objektum alapú szkriptnyelv, szintaxisa hasonló a Java programozási nyelvhez) Számításokon alapuló funkciókat, és a szövegges tartalom feldolgozását szolgálja. A JavasScript teljes értékű programozási nyelvvé fejlődött, amely lehetővé teszi adatok bekérését a felhasználótól, matematikai feladatok és szöveges állományokkal való műveletek elvégzásét a weboldalon. A többi programozási nyelvez hasonlóan különféle adattípusokkal, operátorokkal, változókkal, ciklusokkal és függvényekkel dolgozik. JavaScript kódot tartalmazhat a HTML állomány is, a <script> ...JavaScript kód...</script> tag-ek között. A fejlesztői irányelvek szerint azonban a JavaScript kódot a CCS állományhoz hasonlóan a weboldal gyökérkönyvtárában, egy JS nevű mappa .js kiterjesztésű szöveges fáljaiban tároljuk.

A böngészőprogramok együtt tudják kezelni mindhárom szabvány kódjait, a hozzájuk tartozó képekkel együtt. A weboldalak JavaScript helyett, vagy azzal együtt más programozási nyelveken írt algoritmusokat is használhatnak (PHP, Java, .NET, Pearl, stb).
